# Airport Dashboard Project

## Features
- **Real-time Data**: Displays up-to-date information about flight arrivals and departures.
- **User-Friendly Interface**: Intuitive design for easy navigation.
- **Customizable Views**: Options to view specific airlines, destinations, or time frames.

## Usage Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Whitebeto/dashboard-projecao-aeroporto.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dashboard-projecao-aeroporto
   ```
3. Open the `index.html` file in your web browser to start using the dashboard.

## Design Details
- The dashboard is designed with responsiveness in mind, ensuring it works well on desktops, tablets, and mobile devices.
- Colors and fonts can be easily modified through the `styles.css` file.

## Customization Tips
- To add new features, modify the `script.js` file.
- For additional styling, edit `styles.css`.
- You can update the data sources within the JavaScript file to connect to live data feeds.
